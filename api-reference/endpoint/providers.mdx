## Response

The response contains information about all available cloud providers and their supported features.

<ResponseField name="[provider_key]" type="object">
  Information about a specific cloud provider.
  
  <Expandable title="Provider Properties">
    <ResponseField name="provider" type="string">
      The display name of the cloud provider.
      
      Example: `"AWS"`
    </ResponseField>
    
    <ResponseField name="predicates" type="object">
      Available filtering options and supported values for this provider.
      
      <Expandable title="Predicates Properties">
        <ResponseField name="regions" type="array">
          List of available regions for this provider.
          
          Example: `["us-east-1", "ap-northeast-2", "..."]`
        </ResponseField>
        
        <ResponseField name="usage_types" type="array">
          List of supported usage types (e.g., on-demand, reserved).
          
          Example: `["RESERVED", "ONDEMAND", "..."]`
        </ResponseField>
        
        <ResponseField name="availability_zones" type="array">
          List of available availability zones for this provider.
          
          Example: `["us-east-1a", "us-east-1b", "..."]`
        </ResponseField>
        
        <ResponseField name="providers" type="array">
          List containing the provider name (used for consistency).
          
          Example: `["AWS"]`
        </ResponseField>
        
        <ResponseField name="operating_systems" type="array">
          List of supported operating systems for this provider.
          
          Example: `["Linux", "Windows"]`
        </ResponseField>
        
        <ResponseField name="services" type="array">
          List of cloud services offered by this provider.
          
          Example: `["AmazonEC2"]`
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Supported Providers

The API currently supports the following cloud providers:

- **AWS** - Amazon Web Services
- **Azure** - Microsoft Azure
- **DigitalOcean** - DigitalOcean Droplets
- **GCP** - Google Cloud Platform
- **Hetzner** - Hetzner Cloud
- **OVHCloud** - OVH Public Cloud
